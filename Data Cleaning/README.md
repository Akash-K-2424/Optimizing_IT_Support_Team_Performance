# Data Cleaning Documentation

This folder contains the data cleaning documentation and Python scripts used for preprocessing the IT support team performance data.

## Files

- **data_cleaning.py** - Main Python script for data cleaning operations
- **cleaning_report.md** - Detailed report of cleaning operations performed

## Data Cleaning Steps

1. **Data Loading**: Import raw data from CSV/Excel files
2. **Missing Value Handling**: Identify and handle missing values
3. **Data Type Conversion**: Convert columns to appropriate data types
4. **Duplicate Removal**: Identify and remove duplicate records
5. **Outlier Detection**: Detect and handle outliers in numerical columns
6. **Data Validation**: Validate data integrity and consistency
7. **Feature Engineering**: Create new features for analysis
8. **Data Export**: Export cleaned data for further analysis

## Python Libraries Used

- pandas - Data manipulation and analysis
- numpy - Numerical computations
- matplotlib - Data visualization
- seaborn - Statistical data visualization

## Usage

```python
python data_cleaning.py
```

This will process the raw data and generate cleaned data in the Data/processed folder.
