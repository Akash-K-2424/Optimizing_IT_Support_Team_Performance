# Processed Data

This folder contains cleaned and processed data files ready for analysis and visualization.

The processed data files are generated by running the data cleaning script from the `Data Cleaning` folder.

Expected files:
- it_support_cleaned.csv
- performance_metrics_processed.csv
- aggregated_kpis.csv

## Processing Steps Applied

1. Missing value imputation
2. Duplicate removal
3. Outlier handling
4. Data type conversion
5. Feature engineering
6. Data validation
