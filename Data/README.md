# Data Folder

This folder contains both raw and processed data for the IT Support Team Performance Optimization project.

## Structure

- **raw/**: Contains the original, unprocessed data files
- **processed/**: Contains cleaned and preprocessed data ready for analysis

## Data Sources

The data in this folder includes:

### Raw Data
- IT support ticket data
- Team member performance metrics
- Response and resolution time data
- Customer satisfaction ratings
- Workload distribution data

### Processed Data
- Cleaned and validated IT support data
- Aggregated performance metrics
- Engineered features for analysis

## File Formats

- CSV files for tabular data
- Excel files for data with multiple sheets
- JSON files for structured configuration data

## Usage

1. Place raw data files in the `raw/` folder
2. Run the data cleaning script from the `Data Cleaning` folder
3. Processed data will be generated in the `processed/` folder
4. Use processed data for Power BI analysis and visualization

## Data Privacy

Please ensure that all data stored in this folder complies with:
- Data privacy regulations
- Organizational data policies
- Anonymization requirements for sensitive information
